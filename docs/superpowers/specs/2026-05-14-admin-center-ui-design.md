# 规格说明：优化“管理中心”显示效果 (方案 B)

## 1. 目标
优化顶部导航栏中“管理中心”工作空间项的视觉表现，使其更具现代化质感，并解决位置过于靠左的拥挤感。

## 2. 视觉设计细节 (方案 B: 现代化渐变)

### 2.1 布局调整
- **位置偏移**：为“管理中心”项增加 `margin-left: 12px` (或 `ml-3` Tailwind 类)，使其与侧边栏边缘保持呼吸感。
- **内边距与间距**：
  - 容器增加 `padding: 4px 8px` 以扩大点击区域。
  - 图标与文字间距保持 `gap-3`。

### 2.2 图标样式
- **背景色**：使用 `linear-gradient(135deg, #fb7185 0%, #e11d48 100%)`。
- **圆角**：保持 `rounded-lg` (8px)。
- **投影**：增加彩色投影 `box-shadow: 0 4px 10px rgba(225, 29, 72, 0.3)`。
- **字体**：加粗 `font-bold`，字号 `text-sm`。

### 2.3 文字与交互
- **文字**：颜色保持 `var(--text-primary)`，字号 `text-sm`，字重 `font-bold`。
- **悬停效果**：
  - 整体容器悬停时 `scale: 1.02`。
  - 图标投影加深。
  - 渐变色略微变亮。

## 3. 技术方案

### 3.1 修改 `src/layouts/MainLayout.vue`
- 在工作空间切换器的 `div` 容器上添加动态样式/类，判断 `workspaceStore.isAdminWorkspace`。
- 为图标 `div` 注入特定的渐变色样式。

### 3.2 修改 `src/stores/workspace.ts`
- 确保 `admin-workspace` 的元数据支持这些样式定义（可选，也可直接在 UI 层处理）。

## 4. 验证标准
- [ ] “管理中心”显示时，左侧有明显的 12px 间距。
- [ ] 图标背景呈现红粉渐变色，带有明显的彩色阴影。
- [ ] 悬停时有平滑的缩放和阴影变化反馈。
- [ ] 非管理员工作空间（个人/团队）保持原样，不受影响。
