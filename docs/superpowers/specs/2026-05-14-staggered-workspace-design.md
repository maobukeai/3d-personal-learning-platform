# 规格说明：全局工作空间错位移动与样式优化

## 1. 目标
优化顶部导航栏中工作空间切换器的视觉表现与交互体验。通过为不同类型的工作空间设置不一致的左边距，配合过渡动画，实现在切换工作空间时的平滑横向滑动效果。同时提升“管理中心”的视觉优先级。

## 2. 交互与设计细节

### 2.1 错位布局 (Staggered Positions)
根据当前工作空间类型，动态调整容器的左边距 (`margin-left`)：
- **个人空间 (Personal)**：`ml-4` (16px)
- **团队空间 (Team)**：`ml-10` (40px) —— *微调以增加位移感*
- **管理中心 (Admin)**：`ml-20` (80px) —— *显著区分，突出管理权限*

### 2.2 动画效果
- 容器应用 `transition-all duration-500 cubic-bezier(0.34, 1.56, 0.64, 1)`。
- 使用弹性过渡函数，使滑动过程更具动感。
- 鼠标悬停时保持 `scale-[1.02]` 的轻微放大反馈。

### 2.3 管理中心特殊样式 (方案 B)
当切换到“管理中心”时：
- **图标背景**：`linear-gradient(135deg, #fb7185 0%, #e11d48 100%)`。
- **投影**：`box-shadow: 0 4px 12px rgba(225, 29, 72, 0.3)`。
- **文字**：应用 `tracking-wide` (增加字间距)。
- **箭头**：变为 `text-rose-400`。

## 3. 技术方案

### 3.1 逻辑实现
在 `MainLayout.vue` 中，利用 `computed` 属性或模板内的三元表达式，根据 `workspaceStore.currentWorkspace.type` 动态绑定 CSS 类和内联样式。

### 3.2 代码位置
- 修改 `src/layouts/MainLayout.vue` 模板中的工作空间切换器部分。

## 4. 验证标准
- [ ] 切换到个人空间：容器靠左，图标为默认主题色。
- [ ] 切换到团队空间：容器平滑滑动至中间位置。
- [ ] 切换到管理中心：容器平滑滑动至最右侧，且图标呈现渐变与阴影效果。
- [ ] 动画过程平滑，无闪烁或突变。
